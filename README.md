# Fusca Maragato Interactive Landing Page

Landing page desenvolvida com HTML5, CSS3 e JavaScript, dedicada ao universo do Fusca Maragato. O projeto foi criado para praticar desenvolvimento front-end, organização de código e interação com o usuário através de formulários e elementos dinâmicos.

## 📋 Sobre o Projeto

Este projeto simula uma página promocional para entusiastas de Fusca, apresentando informações sobre a comunidade, imagens do veículo e recursos interativos para aumentar o engajamento do usuário.

Durante o desenvolvimento foram aplicados conceitos de estruturação semântica, estilização moderna e manipulação de elementos da página utilizando JavaScript.

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

## ✨ Funcionalidades

* Página inicial responsiva
* Menu de navegação
* Galeria de imagens
* Formulário de interação com o usuário
* Exibição dinâmica de elementos da página
* Efeitos visuais com CSS
* Organização de conteúdo utilizando Flexbox
* Estrutura de arquivos separada em HTML, CSS e JavaScript

## 🎯 Conceitos Praticados

* HTML semântico
* Flexbox
* Responsividade
* Eventos JavaScript
* Manipulação do DOM
* Formulários HTML
* Organização de projetos front-end

## 📚 Objetivo

O objetivo deste projeto foi aprofundar meus conhecimentos em desenvolvimento web, explorando não apenas a construção visual de uma landing page, mas também a criação de interações que tornam a experiência do usuário mais dinâmica.
